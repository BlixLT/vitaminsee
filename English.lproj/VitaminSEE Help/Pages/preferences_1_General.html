General Preferences

Smooth Images: This option controls the amount of smoothing applied to the Image. "No Smoothing" disables smoothing. This is the fastest setting, but results in the lowest quality. "Low Smoothing" and "High Smoothing" display the image twice--first without smoothing for speed and then secondly with smoothing for final display.

Display Thumbnails: If this option is checked, thumbnails that have already been created will be displayed. If it is disabled, the default file icon will be displayed.

Generate Thumbnails: If this option is checked, thumbnails will be generated and used for images that don't have them. Otherwise, thumbnails aren't generated.
