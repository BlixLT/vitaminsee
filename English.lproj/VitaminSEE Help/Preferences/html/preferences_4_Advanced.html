Advanced Preferences

Preload Images: This option controls whether VitaminSEE will load the next image into memory while displaying the current image. Usually, this improves speed, but on computers with little free memory, it can slow things down while browsing through folders that mainly contain large (greater then 1MB) images. It is checked by default.

Show Hidden Files: Selecting this option will show hidden files. It will show system files in root files, and "dot" files in all other directories.
